Frequently Asked Questions

What is a PWA?

A Progressive Web App is a website that works like a native app. It can be installed on your device and work offline.

Will it work offline?

Yes! Once installed, basic features work offline. AI features require internet connection.

How do I update it?

PWAs update automatically when you're online. Just refresh the app if prompted.
